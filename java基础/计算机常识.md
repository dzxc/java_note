# 人机交互 
软件的出现实现了人与计算机之间的更好的交互
### 交互方式：
•  图形化界面：这种方式简单直观，使用者易于
接受，容易上手操作。

•  命令行方式：需要有一个控制台，输入特定的
指令，让计算机完成一些操作。较为麻烦，需
要记录住一些命令。

### 命令行方式
dos命令行，课程中常见的命令。
•  dir : 列出当前目录下的文件以及文件夹
•  md : 创建目录
•  rd : 删除目录
•  cd : 进入指定目录
•  cd.. : 退回到上一级目录
•  cd/ : 退回到根目录
•  del : 删除文件
•  exit : 推出dos命令行
